**Severity**: medium
**Location**:
```typescript
  const handleScan = async () => {
    if (!url.trim()) return

    abortRef.current?.abort()

        signal,
      })

      if (!res.ok) {
        const data = await res.json()
        if (!signal.aborted) setError(data.error || 'Failed to scan repository')
        return

      if (!signal.aborted) {
        setScanResult(data)
        onSkillsFound(data.skills, url.trim(), ref || 'main')

      if (err instanceof DOMException && err.name === 'AbortError') return

    }
```

**Fix**: Ensure the `ref` value sent to the API is consistent with the fallback logic used elsewhere. Modify the `body` of the fetch request to use `ref || 'main'`.
