### BUG: Potential Race Condition in `AgentRegistry.getAgent`
**Severity**: high
**Location**: `AgentRegistry.getAgent`
**Problem**: The `AgentRegistry.getAgent` method first checks if an agent exists (`this.agents.get(agentId)`). If not, it proceeds to `await this.evictIfNeeded()` and then creates and initializes a new agent. If multiple concurrent calls to `getAgent` for the same `agentId` occur when the agent is not yet in the registry, they will all pass the initial `if (agent)` check. Each call will then proceed to evict if needed and then attempt to create and initialize a new agent. This can lead to multiple instances of the same agent being initialized and potentially added to the `this.agents` map, or at least redundant initialization work. The `this.agents.set(agentId, agent)` call would overwrite previous attempts, but the `await agent.initialize()` would still be executed multiple times for the same logical agent.
**Fix**: Implement a mechanism to ensure that only one initialization process occurs for a given `agentId` at a time. This can be done by using a temporary `Promise` cache for agents currently being initialized.

```typescript
class AgentRegistry {
  private agents = new Map<string, Agent>()
  private initializingAgents = new Map<string, Promise<Agent>>() // New: Track agents currently being initialized
  private accessOrder: string[] = []  // Most recently accessed at the end
  private maxAgents: number

  constructor(maxAgents = 10) {
    this.maxAgents = maxAgents
    console.log(`[AgentRegistry] Initialized with max ${maxAgents} agents (LRU eviction enabled)`)
  }

  /**
   * Update access order (move to end = most recently used)
   */
  private touch(agentId: string): void {
    const index = this.accessOrder.indexOf(agentId)
    if (index !== -1) {
      this.accessOrder.splice(index, 1)
    }
    this.accessOrder.push(agentId)
  }

  /**
   * Evict least recently used agent if at capacity
   */
  private async evictIfNeeded(): Promise<void> {
    while (this.agents.size >= this.maxAgents && this.accessOrder.length > 0) {
      const lruAgentId = this.accessOrder.shift()!
      // Ensure we don't evict an agent that is currently being initialized
      if (this.initializingAgents.has(lruAgentId)) {
        this.accessOrder.push(lruAgentId); // Put it back at the end if it's being initialized
        continue;
      }
      const agent = this.agents.get(lruAgentId)
      if (agent) {
        console.log(`[AgentRegistry] Evicting LRU agent ${lruAgentId.substring(0, 8)} (${this.agents.size}/${this.maxAgents})`)
        try {
          await agent.shutdown()
        } catch (err) {
          console.error(`[AgentRegistry] Error shutting down evicted agent:`, err)
        }
        this.agents.delete(lruAgentId)
      }
    }
  }

  /**
   * Get or create an agent
   */
  async getAgent(agentId: string, config?: AgentConfig): Promise<Agent> {
    let agent = this.agents.get(agentId)

    if (agent) {
      // Update access order (touch = mark as recently used)
      this.touch(agentId)
      return agent
    }

    // If an agent is already being initialized, wait for it
    if (this.initializingAgents.has(agentId)) {
      return this.initializingAgents.get(agentId)!;
    }

    // Evict LRU agent if at capacity before creating new one
    await this.evictIfNeeded()

    // Create new agent
    console.log(`[AgentRegistry] Loading agent ${agentId.substring(0, 8)} (${this.agents.size + 1}/${this.maxAgents})`)
    agent = new Agent({
      agentId,
      workingDirectory: config?.workingDirectory

    const initializationPromise = agent.initialize().then(() => {
      this.agents.set(agentId, agent!);
      this.touch(agentId);
      this.initializingAgents.delete(agentId); // Remove from initializing map on success
      return agent!;
    }).catch(error => {
      this.initializingAgents.delete(agentId); // Remove from initializing map on failure
      throw error;
    });

    this.initializingAgents.set(agentId, initializationPromise);
    return initializationPromise;
  }

    if (agent) {

    }
    return agent
  }

   * Shutdown an agent

    if (agent) {
      await agent.shutdown()
      this.agents.delete(agentId)
      const index = this.accessOrder.indexOf(agentId)
      if (index !== -1) {
        this.accessOrder.splice(index, 1)
      }
    }
    // Also remove from initializingAgents if it was there
    this.initializingAgents.delete(agentId);
  }

  /**
   * Shutdown all agents
   */
  async shutdownAll(): Promise<void> {
    console.log('[AgentRegistry] Shutting down all agents...')
    const shutdownPromises = Array.from(this.agents.values()).map(agent => agent.shutdown())
    await Promise.all(shutdownPromises)
    this.agents.clear()
    this.accessOrder = []
    this.initializingAgents.clear(); // Clear any pending initializations

  }

  /**
   * Get all active agents (currently in memory)

  }

      maxAgents: this.maxAgents,

  }

      }
    }

    return {

      lastMemoryResult,

      agents: subconsciousStatuses
    }
  }
}
```
