Conversation History
Explore What Your Agents Are Doing

Browse, search, and analyze every conversation your AI agents have. See their thinking, understand their decisions, and track their work with powerful search and visualization tools.

Explore Features Get Started

Complete Conversation Visibility

Every conversation, every message, every thought—organized and searchable

WorkTree Browser

Hierarchical view of all agent conversations organized by project, category, and session. Navigate your entire conversation history at a glance.

Chat View

Beautiful chat-style message bubbles that make conversations easy to read. User messages on the right, agent responses on the left, just like messaging apps.

Thinking Messages

See Claude's extended thinking process. Understand how the agent reasons, plans, and makes decisions before responding.

Live Search

Instant search across all messages with text highlighting. Navigate between matches with Previous/Next buttons and auto-scroll to results.

Tool Usage Tracking

See every tool the agent uses—file reads, writes, bash commands, web searches. Expandable tool details show inputs and outputs.

Conversation Metadata

Track git branch, Claude model version, working directory, timestamps, and total message count. Know the context of every conversation.

How It Works

Access conversation history in three simple ways

1

Open WorkTree Tab

Click the "WorkTree" tab for any agent session. See all conversations organized by project hierarchy.

2

Select a Conversation

Click any conversation to view its full details. Toggle between List and Chat views for different perspectives.

3

Search & Explore

Use the search bar to find specific content. Navigate matches, view thinking, and expand tool details.

Why Browse Conversation History?

Essential for understanding and improving AI agent work

Debug Agent Decisions

When an agent makes an unexpected decision, review the conversation to see what context led to that choice. Search for specific variables, functions, or files to understand the agent's reasoning.

Learn Agent Patterns

Study how successful agents approach problems. See which tools they use, how they break down tasks, and what thinking processes lead to good outcomes.

Track Project Progress

See the evolution of a project over time. Find when specific features were implemented, bugs were fixed, or architectural decisions were made.

Onboard Team Members

New team members can browse agent conversations to understand project context, coding standards, and decision history without interrupting others.
