### BUG: Inconsistent `enabledCount` update in `togglePlugin`
**Severity**: medium
**Location**: `togglePlugin` function
**Problem**: When a plugin's enabled status is toggled, the `enabledCount` for the marketplace is recomputed by filtering the `updatedPlugins` array. This is good. However, the `setTotals` call updates `prev.enabledPlugins` by simply adding or subtracting 1 (`prev.enabledPlugins + (currentEnabled ? -1 : 1)`). This approach is prone to errors if `prev.enabledPlugins` was already out of sync due to a previous optimistic update or if the server response was delayed/failed. The `totals.enabledPlugins` should ideally be recomputed based on the actual state of all plugins across all marketplaces, similar to how `enabledCount` is recomputed for individual marketplaces.

**Fix**: Instead of optimistically updating `totals.enabledPlugins` by `+1` or `-1`, recompute it by iterating through all marketplaces and their plugins after the local state has been updated.

Modify the `togglePlugin` function:
```typescript
  const togglePlugin = async (key: string, currentEnabled: boolean) => {
    setToggling(key)
    try {
      const res = await fetch('/api/settings/global-plugins', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ key, enabled: !currentEnabled }),
      })
      if (res.ok) {
        setMarketplaces(prev => {
          const updatedMarketplaces = prev.map(m => {
            const updatedPlugins = m.plugins.map(p => p.key === key ? { ...p, enabled: !currentEnabled } : p)
            return {
              ...m,
              plugins: updatedPlugins,
              enabledCount: updatedPlugins.filter(p => p.enabled).length,
            }
          })
          // Recompute total enabled plugins from the updated state
          const newTotalEnabledPlugins = updatedMarketplaces.reduce((acc, m) => acc + m.enabledCount, 0)
          setTotals(prevTotals => ({ ...prevTotals, enabledPlugins: newTotalEnabledPlugins }))
          return updatedMarketplaces
        })
      }
    } catch (err) { console.error('Failed to toggle plugin:', err) }
    finally { setToggling(null) }
  }
```
