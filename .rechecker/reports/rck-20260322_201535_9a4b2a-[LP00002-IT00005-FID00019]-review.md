### BUG: Potential Race Condition in `evictStaleBuildResults` and `buildPlugin`
**Severity**: medium
**Location**: `evictStaleBuildResults`, `buildPlugin`
**Problem**: The `evictStaleBuildResults` function is called at the beginning of `buildPlugin` to clean up old builds. However, `evictStaleBuildResults` also removes build directories from the filesystem. If a build is currently in progress (i.e., `runBuild` is executing) and its `createdAt` timestamp makes it eligible for eviction, `evictStaleBuildResults` might delete its working directory (`buildDir`) while `runBuild` is still trying to access or write to it. This could lead to `runBuild` failing with "No such file or directory" errors or other unexpected behavior.
**Fix**: When evicting builds, `evictStaleBuildResults` should only remove builds that are *not* currently in a 'building' state. This requires adding a check to ensure that a build is not active before attempting to delete its directory.

```typescript

function evictStaleBuildResults(): void {
  const now = Date.now()
  const activeBuildIds = new Set<string>();
  for (const [id, result] of buildResults) {
    if (result.status === 'building') {
      activeBuildIds.add(id);
    }
  }

  for (const [id, result] of buildResults) {
    const age = now - new Date(result.createdAt).getTime()
    if (age > BUILD_TTL_MS && !activeBuildIds.has(id)) { // Add check for active builds
      buildResults.delete(id)
      // Best-effort cleanup of build directory
      const buildDir = path.join(BUILDS_DIR, id)
      fs.rm(buildDir, { recursive: true, force: true }).catch(err => {
        console.error(`Failed to remove build directory ${id}:`, err)
      })
    }
  }

  // If still over limit, evict oldest
  if (buildResults.size > MAX_BUILD_RESULTS) {
    const entries = [...buildResults.entries()]
      .sort((a, b) => new Date(a[1].createdAt).getTime() - new Date(b[1].createdAt).getTime())
    const toRemove = entries.filter(([id]) => !activeBuildIds.has(id)).slice(0, entries.length - MAX_BUILD_RESULTS) // Filter out active builds
    for (const [id] of toRemove) {
      buildResults.delete(id)
      const buildDir = path.join(BUILDS_DIR, id)
      fs.rm(buildDir, { recursive: true, force: true }).catch(err => {
        console.error(`Failed to remove build directory ${id} during eviction:`, err)
      })
    }
  }
}
```
