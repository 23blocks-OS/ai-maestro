### BUG: Inconsistent `queryOverrides` in `useEffect` dependency array
**Severity**: medium
**Location**: `useDeviceType`
**Problem**: The `queryOverrides` variable is captured once when the `useDeviceType` hook is called. However, `useEffect` has an empty dependency array (`[]`). This means that if `window.location.search` changes (e.g., due to client-side navigation) after the initial render, the `queryOverrides` value inside the `useEffect` closure will become stale. The `update` function and subsequent calls to `detectTouch` and `classify` will continue to use the old `forceTouch` and `forceMobile` values, leading to incorrect device type detection until the component is re-mounted. The comment above `queryOverrides` explicitly mentions this potential issue but the `useEffect` dependency array does not account for it.
**Fix**: Include `queryOverrides` in the `useEffect` dependency array. This will cause the effect to re-run and re-capture the latest `queryOverrides` values whenever `window.location.search` (which `getQueryOverrides` implicitly depends on) changes.

```typescript
// ... (rest of the code)

export function useDeviceType(): DeviceInfo {
  const queryOverrides = getQueryOverrides()

  const [info, setInfo] = useState<DeviceInfo>(() => {
    if (typeof window === 'undefined') return { deviceType: 'desktop', isTouch: false }
    const { forceTouch, forceMobile } = queryOverrides
    const isTouch = forceTouch || detectTouch()
    const deviceType = forceMobile ? 'phone' : classify(window.innerWidth)
    return { deviceType, isTouch }
  })

  useEffect(() => {
    const { forceTouch, forceMobile } = queryOverrides // This will now be fresh if queryOverrides changes

    const update = () => {
      const isTouch = forceTouch || detectTouch()
      const deviceType = forceMobile ? 'phone' : classify(window.innerWidth)
      setInfo(prev => {
        if (prev.deviceType === deviceType && prev.isTouch === isTouch) return prev
        return { deviceType, isTouch }
      })
    }

    window.addEventListener('resize', update)

    const mql = window.matchMedia?.('(pointer: coarse)')
    const mqlAny = window.matchMedia?.('(any-pointer: coarse)')
    if (mql?.addEventListener) {
      mql.addEventListener('change', update)
    }
    if (mqlAny?.addEventListener) {
      mqlAny.addEventListener('change', update)
    }

    update()

    return () => {
      window.removeEventListener('resize', update)
      if (mql?.removeEventListener) {
        mql.removeEventListener('change', update)
      }
      if (mqlAny?.removeEventListener) {
        mqlAny.removeEventListener('change', update)
      }
    }
  }, [queryOverrides]) // Add queryOverrides to the dependency array

  return info
}
```
