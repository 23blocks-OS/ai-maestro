Configure Your Agents

Customize each agent's settings, appearance, and behavior

Display Settings

  • Display Name: Friendly name shown in the dashboard
  • Alias: Short identifier for messaging and CLI
  • Description: Document what this agent is for

Working Environment

  • Working Directory: Default path for the agent's terminal
  • Git Repositories: Link repos for automatic detection
  • Host Assignment: Run on local or remote workers

Portable Agents

Export, transfer, clone, and backup your AI agents across any machine

Export Agents

Export any agent to a portable .zip file. Includes configuration, memory database, message history, and git repository references.

Agent Settings → Export → Downloads .zip file

Import Anywhere

Drop .zip files into AI Maestro on any machine. Automatically restores configuration, memory, and messages.

New Host → Import Agent → Select file → Restored!

Clone Agents

Create copies of agents on the same machine or across hosts. Perfect for spinning up multiple similar agents.

Export → Import on same machine → New clone

Backup & Restore

Keep backups of your critical agents. Restore from any point in time. Perfect for disaster recovery.

Weekly exports → Cloud storage → Instant recovery
